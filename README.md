# Rat-Run
 Stress inducing games for biomarker research about a rat who wants to escape from space sewers

 ## Content
Contains the two different versions of the Rat Run game and the Bride-of-Frankensystem framework integration for running the biomarker study online 

For more info on how to run the study using BOF, see https://github.com/colbyj/bride-of-frankensystem
